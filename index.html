VC
BS-VChanger
Windows Desktop App — Real-time voice changer

Real-time voice changer with 27 built-in profiles and 17 DSP effects, including RNNoise-based noise suppression, pitch shifting, and reverb. Routes processed audio to speakers/headphones and to VB-CABLE for use in Teams, Discord, Zoom, or OBS. Ships as a single ~5 MB standalone EXE — no installer.

BT
BagPipes TikTok Auto
Windows Desktop App — Video pipeline

Crops horizontal Fortnite gameplay clips to 9:16 vertical format using ffmpeg and uploads them to TikTok automatically via the Content Posting API. Built with Rust and egui; OAuth 2.0 with PKCE handles TikTok authentication.
